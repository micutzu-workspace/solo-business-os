# MVP Structure — Solo Business OS

## 1. Dashboard
- weekly focus
- urgent follow-ups
- active leads
- open offers
- top tasks

## 2. Lead Tracker
Fields:
- lead name
- business
- source
- need
- status
- next action
- follow-up date
- estimated value
- notes

## 3. Offer Pipeline
Fields:
- prospect
- offer type
- quoted value
- status
- sent date
- next step
- final result
- lost reason

## 4. Weekly Planner
- top 3 outcomes
- sales tasks
- delivery tasks
- admin tasks
- backlog
- weekly review

## 5. Content Tracker
Fields:
- idea
- format
- channel
- status
- CTA
- repurpose notes
- publish date

## 6. SOP Library
Short SOPs:
- capturing a lead
- qualifying quickly
- following up
- sending an offer
- planning a week
- tracking content
