# MVP QA Checklist — Solo Business OS

## Product quality
- [ ] Dashboard is understandable at first glance
- [ ] Every database has the minimum useful fields
- [ ] Views support daily use, not just visual polish
- [ ] SOPs are short and actionable
- [ ] Setup guide is readable
- [ ] Example data feels realistic

## Scope discipline
- [ ] No unnecessary features
- [ ] No automation dependency in version 1
- [ ] No unsupported claims in docs or landing copy

## Buyer usability
- [ ] Setup can be completed quickly
- [ ] Product is useful for a solo operator without coaching
- [ ] Product promise matches deliverable reality
