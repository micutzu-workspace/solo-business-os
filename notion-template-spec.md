# Notion Template Spec — Solo Business OS MVP

## 1. Dashboard
### Purpose
Main operating view for the user.

### Sections
- weekly focus
- urgent follow-ups
- open offers
- active leads
- content queue
- quick links to SOPs

## 2. Lead Tracker
### Fields
- lead name
- business / client
- source
- status
- next action
- follow-up date
- estimated value
- notes

### Suggested views
- all leads
- active follow-ups
- won / lost

## 3. Offer Pipeline
### Fields
- prospect
- offer type
- quoted amount
- status
- sent date
- next step
- final outcome
- lost reason

### Suggested views
- open offers
- sent this month
- won
- lost

## 4. Weekly Planner
### Fields
- task
- category (sales / delivery / admin / content)
- priority
- due date
- status
- linked lead or offer if relevant

### Suggested views
- this week
- today
- backlog

## 5. Content Tracker
### Fields
- idea title
- channel
- format
- status
- CTA
- publish date
- repurpose notes

### Suggested views
- idea backlog
- in progress
- published

## 6. Mini SOP Library
### Pages
- capture a lead
- qualify quickly
- follow up consistently
- send an offer
- plan the week
- track content simply

## 7. Filled Example
Create one realistic example with:
- a few sample leads
- one or two offers
- a weekly plan
- a few content items
