# Listing Assets Checklist — Solo Business OS

## Product page essentials
- [ ] final product title
- [ ] short subtitle
- [ ] clear hero description
- [ ] bullet list of what is included
- [ ] price set to 19 EUR for early version
- [ ] refund policy text added
- [ ] license/usage note available

## Visual assets
- [ ] 3 to 5 screenshots of the template
- [ ] one cover image for the listing
- [ ] screenshots use generic sample data only
- [ ] visuals match the actual delivered product

## Delivery assets
- [ ] template link or packaged file ready
- [ ] quick-start guide included
- [ ] setup checklist included
- [ ] SOP pack included
- [ ] sample workspace cleaned

## Compliance checks
- [ ] no misleading claims
- [ ] no client or personal data in screenshots
- [ ] payout details verified by account owner
- [ ] tax details reviewed on selling platform

## Launch assets
- [ ] 2 to 3 launch posts ready
- [ ] one short DM/share text drafted for approved direct sharing
- [ ] simple feedback question prepared for early buyers

## Note
This checklist is meant to reduce launch friction and lower avoidable compliance mistakes before the listing goes live.
