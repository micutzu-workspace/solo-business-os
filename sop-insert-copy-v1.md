# SOP Insert Copy — Solo Business OS v1

## Capture a lead
Every real lead should enter the system immediately.

Use this checklist:
- add the lead name
- record the source
- set the current status
- define the next action
- set a follow-up date
- estimate value if possible

## Qualify quickly
Qualification is there to protect your attention.

Use this checklist:
- is the need real?
- is there urgency?
- is the fit acceptable?
- is there realistic value?
- continue, nurture, or archive?

## Follow up consistently
Keep follow-up simple, respectful, and visible.

Use this checklist:
- review due follow-ups daily
- send one clear follow-up
- record the outcome
- define the next step

## Send an offer
Every offer should be visible in the pipeline.

Use this checklist:
- add the prospect
- record the offer type
- record the quoted amount
- record the sent date
- define the next step
- track the final result

## Plan the week
A good week is driven by outcomes, not noise.

Use this checklist:
- define top 3 outcomes
- separate sales, delivery, admin, and content
- keep the active list small
- review before the week ends

## Track content simply
Use content as a business tool.

Use this checklist:
- capture the idea
- assign format and channel
- define the CTA
- update status as it moves
- repurpose strong ideas
