# Template Blueprint — Solo Business OS v1

## Top-level structure
- Home / Dashboard
- Lead Tracker
- Offer Pipeline
- Weekly Planner
- Content Tracker
- SOP Library
- Quick Start
- Setup Checklist
- Example Workspace Guide

## Navigation principle
The default user should be able to understand the structure within 3–5 minutes.

## Home / Dashboard content
### Sections
1. Welcome / short orientation
2. Weekly focus
3. Active leads view
4. Open offers view
5. This week tasks view
6. Content in progress view
7. Quick links to SOP pages

### Dashboard goal
Give a solo operator one page that makes the business feel manageable.

## Supporting pages
### Quick Start
Short explanation of how to begin using the system immediately.

### Setup Checklist
Simple sequence to go from blank template to first useful week.

### Example Workspace Guide
Explains what is sample data and how to replace it.
