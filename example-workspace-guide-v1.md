# Example Workspace Guide — Solo Business OS v1

## Purpose
The example workspace exists to show how the system should feel in actual use.

## What the buyer should do
1. Review the sample items
2. Understand how the dashboard surfaces action
3. Replace the sample data with real business data
4. Archive or delete sample entries not needed

## Important note
The sample data is illustrative. It exists to make the template easier to understand, not to prescribe one single business model.
