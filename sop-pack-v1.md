# SOP Pack — Solo Business OS v1

## 1. Capture a lead
- Add every real lead immediately
- Record source, current need, and next action
- Set a follow-up date

## 2. Qualify quickly
- Check urgency
- Check fit
- Estimate value
- Decide: continue, nurture, or archive

## 3. Follow up consistently
- Review due follow-ups daily
- Send one clear follow-up
- Log the outcome
- Set the next step

## 4. Send an offer
- Add the offer to the pipeline
- Record quoted amount and sent date
- Define next step
- Track outcome

## 5. Plan the week
- Set top 3 outcomes
- Separate sales, delivery, admin, and content
- Keep the active list small

## 6. Track content simply
- Capture ideas in one place
- Set format and channel
- Track publishing status
- Reuse good ideas
