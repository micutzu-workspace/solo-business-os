# SOP Pages — Solo Business OS v1

## SOP: Capture a lead
Every real lead should enter the system immediately.

Checklist:
- add source
- add status
- define next action
- set follow-up date
- estimate value if possible

## SOP: Qualify quickly
Use basic qualification to avoid wasting attention.

Checklist:
- is the need real?
- is there urgency?
- is the fit acceptable?
- is there realistic value?
- continue, nurture, or archive?

## SOP: Follow up consistently
Keep follow-up simple and respectful.

Checklist:
- review due follow-ups daily
- send one clear follow-up
- record the outcome
- set the next step

## SOP: Send an offer
Make every offer trackable.

Checklist:
- record the prospect
- record quoted amount
- record sent date
- define next step
- track final outcome

## SOP: Plan the week
Keep the week outcome-oriented.

Checklist:
- set top 3 outcomes
- separate sales / delivery / admin / content
- avoid overloaded lists
- review before week end

## SOP: Track content simply
Use content as a business tool, not as random activity.

Checklist:
- capture idea
- assign channel
- define CTA
- update status
- repurpose strong ideas
